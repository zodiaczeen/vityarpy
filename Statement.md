# House Price Prediction model Project

# Problem Statement
Valuing residential properties accurately can be challenging for buyers and sellers due to multiple influencing factors. This project aims to automate price estimation by learning patterns from historical property data for a local region.

# Scope of the project
- Works on core features: location, carpet area, bedrooms, balconies, and ownership.
- Trains a regression model to predict sale prices.
- Optimized for fast execution and simple data requirements.

# Target Users
- Home buyers and sellers
- Real estate agents
- Property investment analysts

# High-Level Features
- Accepts and cleans real estate property data
- Automatic handling of missing values
- Efficient training using RandomForest or Linear Regression
- Predicts prices for user-specified property attributes
- Evaluates model using MAE for interpretability
